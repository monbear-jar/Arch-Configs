Use the ``fontextracter.sh`` tool to download, extract, and setup the fonts automatically.

## Dependencies

``7zip``: used to extract

``wine``: runs font installer

## Steps

1. Install dependencies

```
sudo pacman -S 7zip wine
```

2. Grab the link of the iso file from [here](https://archive.org/details/qjwi3h)
3. Run ``fontextracter.sh`` to automatically download and place the fonts in the right directory

That should be all you have to do to get these fonts
